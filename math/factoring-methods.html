Παραγοντοποίηση
Όλες οι Βασικές Μέθοδοι

Δέκα μέθοδοι παραγοντοποίησης με αναλυτικά παραδείγματα και βήματα.

§1

Κοινός παράγοντας

Βρίσκουμε τον μεγαλύτερο κοινό παράγοντα (ΜΚΔ) όλων των όρων και τον βγάζουμε έξω.

Παράδειγμα:

\(12x^3y - 18x^2y^2 + 6x^2y\)

Βήμα 1 — Ο ΜΚΔ των τριών όρων είναι \(6x^2y\):

\(12x^3y = 6x^2y \cdot 2x\)
\(18x^2y^2 = 6x^2y \cdot 3y\)
\(6x^2y = 6x^2y \cdot 1\)

Βήμα 2 — Βγάζουμε τον ΜΚΔ έξω:

\(12x^3y - 18x^2y^2 + 6x^2y = 6x^2y(2x - 3y + 1)\)
§2

Κοινός παράγοντας σε ομάδες (ομαδοποίηση)

Ομαδοποιούμε όρους ώστε να εμφανιστεί κοινός παράγοντας σε κάθε ομάδα.

Παράδειγμα:

\(3x^2 + 6xy + 2x + 4y\)
  • 01

    Χωρίζουμε σε ομάδες: \((3x^2 + 6xy) + (2x + 4y)\)

  • 02

    Κοινός παράγοντας σε κάθε ομάδα: \(3x(x + 2y) + 2(x + 2y)\)

  • 03

    Εμφανίζεται κοινός παράγοντας \((x+2y)\):

\(3x^2 + 6xy + 2x + 4y = (3x + 2)(x + 2y)\)
§3

Διαφορά τετραγώνων

Τύπος: \(\alpha^2 - \beta^2 = (\alpha+\beta)(\alpha-\beta)\)

Παράδειγμα:

\(9x^2 - 16y^2\)

Αναγνωρίζουμε τέλεια τετράγωνα:

\(9x^2 = (3x)^2 \qquad 16y^2 = (4y)^2\) \(9x^2 - 16y^2 = (3x + 4y)(3x - 4y)\)
§4

Άθροισμα κύβων

Τύπος: \(\alpha^3 + \beta^3 = (\alpha+\beta)(\alpha^2 - \alpha\beta + \beta^2)\)

Παράδειγμα:

\(8x^3 + 27\)

Γράφουμε ως τέλειοι κύβοι:

\(8x^3 = (2x)^3 \qquad 27 = 3^3\)

Εφαρμόζουμε τον τύπο με \(\alpha = 2x\), \(\beta = 3\):

\(8x^3 + 27 = (2x+3)(4x^2 - 6x + 9)\)
§5

Διαφορά κύβων

Τύπος: \(\alpha^3 - \beta^3 = (\alpha-\beta)(\alpha^2 + \alpha\beta + \beta^2)\)

Παράδειγμα:

\(x^3 - 125\)

Γράφουμε: \(125 = 5^3\), άρα \(\alpha = x\), \(\beta = 5\):

\(x^3 - 125 = (x-5)(x^2 + 5x + 25)\)
§6

Τέλειο τετράγωνο

Τύπος: \(\alpha^2 \pm 2\alpha\beta + \beta^2 = (\alpha \pm \beta)^2\)

Παράδειγμα:

\(4x^2 + 12x + 9\)

Ελέγχουμε αν είναι τέλειο τετράγωνο:

\(4x^2 = (2x)^2 \qquad 9 = 3^2 \qquad 12x = 2 \cdot 2x \cdot 3\) ✓ \(4x^2 + 12x + 9 = (2x+3)^2\)
Αντίστοιχο με πλην: \(4x^2 - 12x + 9 = (2x-3)^2\)
§7

Τριώνυμο \(x^2 + (\alpha+\beta)x + \alpha\beta\)

Ψάχνουμε δύο αριθμούς \(\alpha, \beta\) τέτοιους ώστε \(\alpha+\beta = S\) και \(\alpha\cdot\beta = P\).

Παράδειγμα:

\(x^2 + 11x + 24\)

Ψάχνουμε \(\alpha, \beta\) με:

  • Άθροισμα \(\alpha + \beta = 11\)
  • Γινόμενο \(\alpha \cdot \beta = 24\)

Οι αριθμοί είναι \(3\) και \(8\)  (\(3+8=11\), \(3\cdot8=24\)).

\(x^2 + 11x + 24 = (x+3)(x+8)\)
§8

Τριώνυμο \(ax^2 + bx + c\)  (\(a \neq 1\))

Χρησιμοποιούμε πολλαπλασιασμό ακραίων + ομαδοποίηση.

Παράδειγμα:

\(2x^2 + 5x - 3\)
  • 01

    Πολλαπλασιάζουμε ακραίους: \(2 \cdot (-3) = -6\).

  • 02

    Ψάχνουμε \(\alpha, \beta\) με γινόμενο \(-6\) και άθροισμα \(+5\): οι αριθμοί είναι \(6\) και \(-1\).

  • 03

    Αναλύουμε τον μεσαίο όρο: \(5x = 6x - x\).

  • 04

    Ομαδοποιούμε: \((2x^2 + 6x) + (-x - 3)\)

  • 05

    Κοινός παράγοντας σε κάθε ομάδα: \(2x(x+3) - 1(x+3)\)

  • 06

    Κοινός παράγοντας \((x+3)\):

\(2x^2 + 5x - 3 = (2x-1)(x+3)\)
Αυτή η μέθοδος λειτουργεί για κάθε τριώνυμο με \(a \neq 1\). Εναλλακτικά μπορούμε να βρούμε ρίζες με διακρίνουσα \(\Delta = b^2 - 4ac\).
§9

Συνδυασμός μεθόδων (ομαδοποίηση + ταυτότητες)

Μερικές φορές χρειάζεται να συνδυάσουμε παραπάνω από μία μέθοδο.

Παράδειγμα:

\(x^3 + 3x^2 - 4x - 12\)
  • 01

    Ομαδοποίηση: \((x^3 + 3x^2) + (-4x - 12)\)

  • 02

    Κοινός παράγοντας σε κάθε ομάδα: \(x^2(x+3) - 4(x+3)\)

  • 03

    Κοινός παράγοντας \((x+3)\): \((x+3)(x^2-4)\)

  • 04

    Το \(x^2-4\) είναι διαφορά τετραγώνων (§3): \((x+3)(x-2)(x+2)\)

\(x^3 + 3x^2 - 4x - 12 = (x+3)(x-2)(x+2)\)
§10

Παραγοντοποίηση με αντικατάσταση (διτετράγωνο)

Όταν η έκφραση είναι τετράγωνο ως προς \(x^2\), θέτουμε \(\omega = x^2\) για απλοποίηση.

Παράδειγμα:

\(x^4 - 5x^2 + 4\)
  • 01

    Θέτουμε \(\omega = x^2\): \(\quad \omega^2 - 5\omega + 4\)

  • 02

    Παραγοντοποιούμε ως τριώνυμο (§7): \((\omega-1)(\omega-4)\)

  • 03

    Επαναφέρουμε \(x\): \((x^2-1)(x^2-4)\)

  • 04

    Και τα δύο είναι διαφορές τετραγώνων (§3):

\(x^4 - 5x^2 + 4 = (x-1)(x+1)(x-2)(x+2)\)
📌 Σημείωση: Το άθροισμα τετραγώνων (π.χ. \(x^2 + 9\)) δεν παραγοντοποιείται στο σύνολο των πραγματικών αριθμών. Στο μιγαδικό πεδίο παραγοντοποιείται, αλλά ξεφεύγει από τη σχολική ύλη.
